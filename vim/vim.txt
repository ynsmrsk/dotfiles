------------------------------------------------------------- MISC ----------------------------------------------------------
gf = open file under the cursor
------------------------------------------------------------- MOVE AROUND ----------------------------------------------------------
^ = Jump to start of line (until whitespace)
- = move up via indent
+ = move down via indent line
{ = Jump backward one paragraph
} = Jump forward one paragraph
( = Jump backward one sentence
) = Jump forward one sentence
H(igh) | M(iddle) | L(ow)
...gg/G = move to n'th line
zz = bring center the current line
zt = bring top the current line
zb = bring bottom the current line
------------------------------------------------------- SEARCH AND SUBSTITUTE -----------------------------------------------------
* = highligt words that are same which under the cursor
:%norm di" = del all strings inside the " on all lines
:s/.../... = substitute a word on the line
:s/.../.../g = substitute each of the words on the line
:%s/.../... = search and substitute each word in the file
:%s/.../.../c = search all wanted words and replace them if you wish
:g/.../s/^/#/g = search pattern go beginning write "#" globally
:%g/.../d = delete lines which contains searched word
:%s/\ ...//g = delete selected word on globally
---------------------------------------------------- WORD-LETTER-PARAGRAPH EDIT --------------------------------------------------
select+= = makes autoindent
number + dd
number + dj/dk
---------------------------------------------------------- YANK-PASTE -----------------------------------------------------------
y2aw = yank 2 all word
V+"+...+y = sel line, choose a register key to yank
"+...+p = call the registered key to copy
p = paste above
------------------------------------------------------- COMMAND LINE -------------------------------------------------------
:norm = implement text to selected line
:sh = start a subshell | C+d / exit = close and get back to vim
:read !ls -a *jpg (meselam)
:norm I<a href" (meselam)
-------------------------------------------------------- INSERT MODE  -------------------------------------------------------
Ctrl+c = go out insmode
Alt+A = go to end of the line
Alt+y/p = yank / paste when insmode
--------------------------------------------------------- WINDOWS ---------------------------------------------------------
(C-w)/z + _ = maximize window
(C-w)/z + = = make equal all windows
(C-w)/z + s = divide vertical
(C-w)/z + v = divide horizontal
--------------------------------------------------------- SURROUND-VIM  ---------------------------------------------------------
ys}... = put paragraph in ...
yssa = put sentence in </>
yswr = put word in [] from the cursor position
yswf = put word in func
----------------------------------------------------------- TERMINAL -------------------------------------------------------------
ZZ = :wq and go back to terminal
C-z = suspend vim | fg = resume
----------------------------------------------------------- INDENT -------------------------------------------------------------
C-t = indent in insert mode
C-d = unindent in insert mode
number + >>/<</>k/<j...
